# Architecture Frameworks
Standardizing how we describe our cloud infrastructure to technical stakeholders.
* **The Stack Model:** Always reference our standard deployment model: Secure Edge -> Central Data Lake -> Unified Microapp Layer.
* **Addressing Security:** Enterprise stakeholders will lead with security questions; have the [Security & Compliance](../playbooks/technical_discovery.md) documentation ready to share.
* **Best Practice:** When proposing a custom solution, ensure it maps to the [Company Standard Architecture](../playbooks/architecture_frameworks.md) to minimize long-term maintenance costs.
